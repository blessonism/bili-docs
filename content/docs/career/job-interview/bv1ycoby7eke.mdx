## 这半个月在干嘛：找到 Java 后端实习，入职一周

距离上一个视频大概半个月。最近就是到处投简历，找到了一个 Java 后端开发岗的实习。这周实习满一周了，也不能说工作，我干那点活还算不上。

这期想说一下我的感受，以及我一路学过来的一些建议。先说明一下，我只是大三，刚找到实习，不算了解整个行业，也不算真正出社会。接下来讲的都是从我个人角度、个人经历和体验出发的，大家听到觉得有用的拿走就行，不用全参考。

## 实习难不难找：Java 很卷，但实习没那么难

先说找实习这事。我感觉 Java 实习其实不是很难找。

网上很多人说 Java 没救了、以后找不到工作。我从实习角度看，Java 这个方向确实很卷，卷的原因是学的人多。但学的人多，也是因为岗位多、需求多。正因为岗位多，你学了东西，大概率是能找到的。

反过来你学 C++、Python，有时候会更没方向，但 Java 网上资源多、路线也多，你至少知道自己该学什么。

## 大三找实习要会多少：会 Spring + 一个项目基本够

从我个人角度，大三找实习不需要会很多东西。我当时的进度是：学完了 Spring；Web 前端略懂一点；微服务学了一些理论。找到实习之前，我还在跟着网上做一个微服务项目。

我简历写得也挺烂的：只写了一个外卖项目，再加上技能写了了解前端、会 Java、会 Spring、会 Git、会 Maven，另外写了“对微服务有一定了解”。就这样也能找到。

## 项目怎么写：苍穹外卖够用，但一定要讲清楚

我写的项目就是苍穹外卖。外卖项目够不够找实习？我觉得完全够。因为你是实习生，能写一些简单的 CRUD 就已经很够了，难的业务大概率也不会交到你手上。

但重点是：你写在简历上，必须能把它讲清楚。你要清楚它用的是什么框架，功能是怎么实现的，比如：

- 怎么提交请求、怎么接收数据请求
- AOP 编程怎么做，用什么注解
- Spring Task：Cron 表达式、定时处理怎么配置
- 拦截器怎么用
- JWT Token 做验证的流程

这些都要理清楚。你只是跟着视频敲一遍，直接拿去投，肯定不够。就算有人愿意面你，他也一定会问你项目。你抄完之后要自己回头捋一遍，或者重新敲一遍，把整个项目理顺。这样你去找那种日常实习（不指望转正那种），基本没问题，够应付面试。

## 入职第一周干啥：熟悉环境 + 新项目上手

这一周大概是这样：

第一天进去主要熟悉环境，配 Git、配 Maven 这些，没给我安排什么活。我就在那敲 LeetCode。

第二天开始，他们因为在做一个新项目，所以就开始有事情了。我也感觉这可能是我进实习比较顺利的原因之一。所以大家面试也别气馁，很多时候真的挺吃运气的。

我自己也投了大概上百家，才有现在这个结果。

## 投简历平台建议：Boss 直聘别指望，其他平台更靠谱

给一些找实习的建议：Boss 直聘我不太建议用，上面很多基本都是已读不回。

如果你要找开发岗，可以用智联、前程无忧、猎聘之类的。至少有的平台还能让你看到简历状态，比如待定、不合适之类的。Boss 直聘很多时候就是海投，但大概率找不到，除非运气特别好。

## 学校资源：能问就问，别等消息晚了

找实习也可以问下学校那边有没有资源，或者问学长学姐。我就挺后悔的：有个离学校很近的实习公司在招人，但消息发得太晚了。要是早一点发到我们学校群里，我可能就去那家了，通勤特别近，坐地铁 20 分钟就到。

我现在这家通勤挺累，每天早上挤成都 1 号线，去双子塔那边，真的很挤。有两次被挤到坐过站两三站，挺破防的。

那个离学校近的机会，其实是我们以前的学长想给我们机会，他联系了系里的老师，但老师一开始拒绝了，理由就是安全第一，不敢放。很多辅导员、老师跟就业这些也有关系，他们更希望你别出事、老老实实待学校里，他们拿死工资就行。

不过我还是挺感谢我辅导员的，他比较支持我。其他一些老师态度就比较模糊，我也能理解，他们也不敢打包票。

## 课能不上就不上：别在学校浪费时间

我的建议是：课能不去上就不去上。就算扣平时分，该走就走，不要在学校里浪费时间。学校学不到什么东西，你该找实习就去找实习。

至于担心挂科，我觉得没必要太担心。平时作业交了，出勤扣点分就扣点分，考试一般也好过。大学考试大家都懂，找题库背题，体现不出学习能力，更多体现的是信息搜集和总结归纳能力。

## 对行业和焦虑的看法：别只看网上，早点去实习体会

再说一下我对计算机行业的看法。这个行业确实可能无聊、枯燥，但也没办法。现在能找到个工作已经不错了。

我对自己也有个期待：以后多打开视野，不要只局限在这一行。多读点书，多学点有用的东西，多出去跟人社交，为以后更多元化的职业做点打算。现在经济就是这样，我倒不觉得有多离谱，只是前 20 年高速发展让大家以为随便就能找到工作。最近这几年可能才更接近正常状态：没有战争，也没有瘟疫，其实挺好的。工作难找一点，但也不是找不到，你必须有对应的技能，有技能才有更好的工作，这很正常。

大家不要焦虑。迷茫很正常，因为现在大学教育确实不太行。但我觉得你迷茫、焦虑反而是好事，至少比在大学里摆烂只会玩的人强。迷茫不知道学什么，就去找人交流，多去看，或者找一个自己感兴趣的方向。哪怕不知道能不能就业，也去学，让自己每天累起来，会好很多。

但不要摆烂。除非你家里有矿，或者你已经想好了，躺平也无所谓。

## 找不到计算机工作怎么办：能养活自己就行

就算你以后担心找不到计算机工作，能干的岗位也多，比如销售、客服，至少能温饱。能活在这世上就已经不错了，没必要那么焦虑。

## 也不是“完全没工作”：身边例子说明学了就有机会

我觉得只要你学了点东西、能干活，应该不难。

我旁边有个前端老哥，比我大两届。他是四川一个二本学校，大数据专业。他跟我说他大一大二就是摆烂，大三寒假开始学，一直学到 8、9 月份秋招，学了整整九个月，每天学四五个小时，中间也会断断续续休息。他投了十多家，拿到了六七个 offer。他说当时还想冲大厂，但挂在算法题上。

所以计算机这行可能确实难，但也不至于难到完全找不到。

## 网上“计算机不行了”的三类人：别被带节奏

我觉得网上说“计算机不行了”的人，大概有几类：

1. 不是计算机专业的，跟着网上说。
2. 以前能拿很高薪水的人，现在可能四五年经验只能拿八九千，他们不愿意干，就说行业不行了。
3. 卖课的，他们会散播焦虑，然后再教你怎么做，让你买他的课。

我的建议是：早点出社会，早点找个实习，用自己的经历去体会，而不是一直在网上看别人说怎么怎么了。没必要焦虑到睡不着、玩不开心。

## 最后说明：都是个人经历，欢迎在评论区交流

我上面说的都只是我个人经历和看法。你的心态、找工作的经历跟我不一样很正常，也可能跟我的观点不一样。也欢迎大家在评论区说说自己的想法，帮大家一起看看以后职业怎么规划、该怎么办。
