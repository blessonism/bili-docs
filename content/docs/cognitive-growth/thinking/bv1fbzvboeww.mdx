## Clawdbot 刷屏引发的社区情绪

大家好，今天我的心情非常低落。昨天 Clawdbot 在推上疯狂刷屏。简单来说，这是一个可以在你的服务器上24小时做各种工作的机器人，所以整个社区都沸腾了，甚至听说引起了 Mac mini 的抢购潮。整个推上都陷入了一种既兴奋又焦虑的情绪，很有意思。今天我想分享一下，在AI时代我们经常遇到的这种情绪，我们应该怎么样处置。如果您能看完这个分享，我相信一定会有所收获。

## 技术不是瓶颈，缺乏场景才是

关于这个工具的爆火，我并没有很惊讶。因为去年11月我就发了一个帖子，我当时认为全世界有点架构能力的人，应该都在做同样的事。我自己不是也做了一个可以开多个容器、并发执行任务的 bot 吗？但是我遇到的瓶颈其实不是技术问题，是因为我所能涉及的行业太少了，我没有场景，没有什么值得我24小时去跑的有价值的任务。我今天又试了一下 Clawdbot，心情沉重，人家这个体验确实做得更好一些。

## AI 时代的 FOMO 情绪与信息源

今天我想聊聊AI时代我们的 FOMO 情绪，和我自己对付它的办法。很多朋友应该还没听过 FOMO 这个词，全称叫 Fear of Missing Out，形容一种害怕错过机会的焦虑情绪。我以前经常听投资人提到这个词，尤其是移动互联网高潮的时候，投资人都害怕错过，所以经常要抢，导致很多泡沫估值。现在这个词越来越多出现在AI行业了。你想想，这就跟外挂一样，你睡觉的时候它帮你干活，而且有无限的 worker 可以同时干活，谁愿意错过这样的机会？谁听了不兴奋？

昨天的感受是，整个推上陷入了兴奋和焦虑两种情绪。兴奋的人跑通了，在晒成果；焦虑的人不知道其他人在高兴什么，但就是焦虑，于是就跑去买 Mac mini。还有一些自媒体，因为这些话题有流量就疯狂炒作。在我看来，这些焦虑和盲目的兴奋，其实都是因为没有从本质理解这件事。尤其是买 Mac 就挺扯的，也不知道谁起的头。改天我会细聊一下AI时代的信息源问题。现在 X（推特）和油管都不是我的信息源，它们最多是让我知道有什么事儿诞生了。但是我们必须有一个手段，能更深层次地理解每一个新事物。

## 用 GITHUBKB 构建本地知识库

我给大家看看我常用的一个 skill，我叫它 GITHUBKB，也就是 knowledge base 的意思。这个技能很简单，只有100行左右的描述，人人都能复刻。简单来说，我的每一台服务器上都有一个叫 GITHUB 的目录，里面就是我的工具链和我关注的技术的代码仓库。这个技能会摘要所有的 repo 和我这个目录，然后把它放在根目录的 clawd.md 里头。这样我就通过这个技能，把目录当做了我的知识库。

你们可以理解，这就是一个非常简单、基于文件系统的 RAG。虽然只有简单的提示，但是借助 `gh` 这个强大的命令，甚至可以把它当做一个搜索引擎来用。能用的场景太多了，比如说帮我们找一下 Clawdbot 和其他工具链的 repo，研究一下怎么部署，看一下它最新合并的 PR 有没有什么更新。我经常用这种功能来了解一个软件最新的动态，这就相当于搜索引擎了。

## 别用农业时代的努力追赶工业时代的速度

我这儿有一个暴论：千万别用农业时代的努力，去追赶工业时代的速度。比如说我想在本地部署一个 Clawdbot，我绝对不会去读文档的。我会通过这个技能直接提问，AI会自动读代码库，按要求来部署。通过简单的问答我就知道了，Clawdbot 其实是可以运行在 Linux 容器上的。我用了一个九代的英特尔 CC150，这几百块钱的CPU，再加一个 fnOS，就轻轻松松地跑了许多容器的实例。所以大家可以看看哪些自媒体在说赶快去抢购 Mac，那么这些自媒体一定不能成为你的信息源。

我让这个 skill 编译容器，顺便再把模型配置成我最近刚弄好的 GLM-4-Flash。虽然遇到了一些配置的问题，但都是可以通过简单的粘贴就能解决。因为AI有整个代码库，所以完全不用人干预。用了没有半个小时，我就全部署好了。而且我还在 Discord 上配了一个机器人，人在外头也可以用它来远程指挥家里的“牛马”来干活。确实 Clawdbot 想象空间挺大的，它通过一个 gateway 节点来控制多个其他的节点，这和我当时做的那个工具是完全一样的。不过它还支持安卓和苹果手机的节点，这个就挺离谱的。

## AI 时代的元能力与核心竞争力

如果你们能看到这里，也许 GITHUBKB 这个技能能带给大家一些思路。用 GitHub 代码库作为本地知识库，去快速扩展自己的能力，这是一种元技能。通过我的这个例子思考一下，如何构建自己在AI时代的元能力，也就是 learn how to learn 的能力。

AI时代的黄金绝对不会是这些 skill 和工具，而是新的场景和商业模式。AI skill 以及如同 Clawdbot 这样的工具，会成为一个个的杠杆，把一份份时间变成许多成果。而把这些成果变成钱，是AI解决不了的。AI时代真正能赢的人，不是技术懂得多的人，而是能把成果转换为钱的人。所以我说了，经历丰富、了解各行各业又不懂技术的普通人，可能才是AI时代最大的受益者。
