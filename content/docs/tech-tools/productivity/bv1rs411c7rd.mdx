## 把软件装到 D 盘，C 盘就不会满吗？
你以为软件装到 D 盘，C 盘就不会满了吗？网上都是这么说的，难道网上说的也有错？

很多软件即使把安装位置改到 C 盘以外的盘，但是软件的数据文件夹默认还是在 C 盘「我的文档」下面。比如聊天记录文件夹就会存在「我的文档」下面，而现在的聊天记录一般都特别大。所以如果你 C 盘比较小，建议把「我的文档」的位置改到 D 盘去。

## 正确迁移“我的文档”到 D 盘的方法
很多人在改文档位置的时候方法错误，会改得乱七八糟。如果需要改文档的位置，可以按这个方法改，一般不会有问题。

对着「文档」点右键，点击「属性」，再点击「位置」。这里可以看到文档默认的保存路径是在 C 盘下面。为了防止大家改错，在改的时候只需要把第一个字母 C 改成 D 就可以了，当然前提是你有 D 盘。

改完之后点击「确定」，再点击「是」，这里两个地方都要点「是」。

## 改完后的效果
这样就更改成功了，既简单又不会出问题。改完之后打开 D 盘，这里会出现一个 user 文件夹，打开这个文件夹就可以看到我们的用户名，文档文件夹就被成功移动到了 D 盘这个位置。

以后如果安装软件，它的所有数据就会自动保存到 D 盘这个位置，就不会占用 C 盘的空间了。
