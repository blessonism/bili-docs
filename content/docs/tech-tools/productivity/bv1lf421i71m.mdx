## 插件系统：构建工具的护城河

Eagle 4.0 带着非常多的优化和插件功能发布了。我的使用感受是 Eagle 真的很有野心。不知道你们有没有想过，插件对于生产力来说意味着什么？工具好不好用，插件至少占了30%。甚至工具的护城河，在一定程度上是靠丰富的插件建立起来的，它会让后来者很难追赶。

## 丰富的格式支持与联动潜力

有了插件，我们终于可以不用靠官方的迭代来升级功能了。之前的 Eagle 论坛里一直有人求各种格式的支持，现在已经有不少开发者开始自己动手了。比如现在已经上线的格式插件，有给建模师的 3D 插件，给剪辑师的视频缩略图插件，给动效师的 Lottie 和 SWF 格式插件，给读书爱好者的 EPS 和 ePub 电子书格式，给开发者的 JSON 格式拓展，最后是给设计师的 TGA 和 SVGA 图像格式拓展。上架的速度还是非常快的。

所谓插件，就是用 Eagle 开放的 API 来做特定的功能，理论上是允许和工具联动的。比如和文生图工具联动，直接把生成的图片收集到你的 Eagle 里；比如链接你信任的图片压缩工具。虽然现在还都没有支持，不过相信官方的接口也会慢慢开放更多能力，到时候只要想象力足够丰富，Eagle 也会越来越强大。先别着急去下载，我来帮你们介绍一下现有的插件，比你们一个个去看效率会高很多。

## 插件分类与检查器功能

Eagle 把插件分为了三种类型：窗口、格式和检查器。

检查器比较简单，就是给媒体增加了更详细的信息。比如对摄影师非常友好的 EXIF 元数据直方图，还有一些媒体的基本信息，再比如图片的尺寸、体积、格式，还有添加日期、创建日期和修改日期。最后是 Eagle 重要的能力：工具窗口。

## 找图神器：以图搜图与一键换大图

目前窗口类的插件又分为了几种：AI 相关、找图相关、格式转化还有其他的。

比如 Beta 版最先上线的 Pinterest 以图搜图，顾名思义用的就是 Pinterest 的服务。现在又加了以图搜图的其他服务，比如 Google、必应等。其实大家平时都在用，但是 Eagle 内置之后，路径会变短很多。以图搜图是一个无敌好用的功能，因为我们经常会有两个问题：第一个是我想要做类似这个风格的作品，但是单一作品很难真正消化；第二个是这个作者的风格我真的很喜欢，我会非常想找到这位作者去关注他。

还有一个功能特别有意思，叫“一键换大图”。它可以帮你从网上搜索更高清的图片资源，来替换掉你的低分辨率图片。这个功能太有意思了，某种程度来说，这才是真正的以图搜图。

## 格式转换与 AI 增强工具

创作工作者们根本离不开格式转化工具、压缩工具还有批量操作，这省掉了买格式转换和压缩工具的钱。更开心的是，它不仅支持了图片和视频，还有视频转 GIF、UTF-8 编码转换，还有 Lottie 的格式转化。相信之后会有更多的格式转化。

AI 相关的功能，比如图片放大，这效果我当时真的是吃了一惊，这高清大图你受得了吗？一键抠图临时使用还是没有问题的，但有的时候抠得不是很好，而且没有办法精修，严格的抠图还得是 PS。AI 橡皮擦现在也是比较初级的阶段，目前也无法替代 PS。不过插件之间还是可以搭配使用的，比如裁切大师加上图片放大，把图片内容裁切出来之后，分辨率会下降得特别厉害，用图片放大可以修复一些问题，接着可以用抠图工具把主体物抠出来。

## 呼吁开发者加入插件生态

如果你有开发能力的话，一定要来尝试制作插件。虽然我靠 AI 尝试过自己开发，但是我的开发水平实在是太烂了，所以失败得很彻底。如果你开发的插件被上架了，这里会显示你的名字。拜托各位大神快来发布你的插件！

## 细节拉满的 UI 与交互优化

除了插件，还有非常多的优化。比如用了更现代化的毛玻璃外观，舒服了很多；增加了批量修改文件；扫描文件的时候还增加了选择，可以选择扫描完全重复的还是扫描相似图片；右键居然还增加了搜索。

快捷键 T 打标签是 Eagle 核心功能之一，4.0 做了性能优化，还给窗口重新做了设计，左侧增加了标签群组的选择功能。如果你有良好的习惯，提前给自己的标签分类的话，效率会提升非常多。

快捷键 F 快速整理图片又优化了，选择图片之后，按 F 键选择你要移动到的文件夹，可以是多个，还可以搜索，非常快。快捷键 J 快速打开文件夹，是给习惯全屏的用户的。Eagle 团队真的太懂交互了。

此外还做了很多的细节优化，比如视频悬停播放响应速度、分享支持 AirDrop、强化了复制的功能、文件夹的缩略图不裁切了、字体预览性能优化（还能看到是否有缺失的字）、浏览器优化等等。非常多的功能和性能优化，我个人是非常建议大家去更新的，而且你放心，更新是免费的。
