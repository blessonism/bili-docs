## Raycast 是什么：桌面上的快捷入口

今天首先介绍的工具叫做 Raycast。它其实很简单，就是桌面上一个快捷入口，可以快速唤起应用，以及各种相关小工具。

用 Mac 的同学可能比较熟悉系统自带的 Spotlight Search，可以唤起应用，但功能相对简单。Mac 上有一款能力很强、用得也非常多的工具，就是 Raycast。

我个人偏好 Raycast，主要原因一方面是功能非常全、非常强；另一方面界面清晰干净，有设计感。

另外，Raycast 原来一直是 Mac 平台，去年终于开始推出 Windows 版本了。大家在官网可以看到，现在已经可以加入 Windows 的 waitlist，并且可以开始试用。也就是说，它可以很好地帮助大家在 Mac 和 Windows 平台上，快速唤起应用和各种小工具，处理各种高效的事务。

## 核心功能概览：剪贴板、AI、表情、计算器、窗口管理

从官方页面展示来看，它有几个核心功能：

- Clipboard History（剪贴板历史）：特别有用。我们经常复制粘贴，会覆盖剪贴板，有一个剪贴板管理工具很必要。类似工具很多，但如果它集成在应用管理器/小工具管理器里，对大家来说是一个很好的统一入口。
- AI 工具：这块是收费的。
- Emoji 表情选择器。
- 内置计算器。
- 窗口管理器。

这一系列小工具全部集成在一个统一入口里，你点开之后就可以管理各种功能操作，这是它很主打的特色：人性化、功能全面。

## 最强的点：Extension Store 插件生态与统一 UI

除了预置功能，Raycast 最强的点是自带商店（Store），里面的插件叫 Extensions。内容非常多，真的包罗万象。

它有个很好的地方不只是插件多，而是给这些插件定义了一套专门的 UI 体系，让大家能在同一个框架里呈现内容。有人可能会觉得这样会不会把内容限定死、不能自由发挥；但自由发挥的结果往往会乱。Raycast 给你特定框架，你在里面创作并套进去，整体一致性非常好。

我很推荐大家体验一下 Raycast 的插件体系，也可以和国内的类似工具对比一下。插件内容很丰富，比如生产力相关的、翻译、Spotify、Arc 等等。一句话，不是每个都好用，确实有很多做得一般，但它的体系比较完整、比较工程化。

一般我的使用习惯是：本地常用的工具，如果希望通过统一入口唤起，我会先去 Store 里搜一下有没有对应的扩展；有的话看看能提供哪些助力，试用一下，不合适随时删。Store 给它本身的能力增加了很多助力。

## 持续演进：AI、专注模式、速记与 Notes

Raycast 本身也在不断演进。最近做得比较多的是 AI 功能，比如可以在聊天框里直接进行 AI 对话（AI Chat），这也是 Pro 主打能力。

另外它还在做一些类似 Zen Mode 的功能，让页面更 focus，减少干扰；也在把速记、Quick Note 这类功能做成核心能力。

整体趋势就是：你以后可能不需要那么多零散的小程序、小工具放在桌面上，入口统一到 Raycast 里。

比如：

- 专注模式（Focus / 类 Zen Mode）。
- 小笔记（Quick Note）：可以直接创建笔记，并支持 Markdown。
- Quick Link：创建快速打开页面的方式。
- Window 管理等。

这些以前都需要找很多小工具来拼凑，现在可以统一在 Raycast 这个平台里使用，方便很多。

## 免费与 Pro：差异点在哪里

非 Pro 版本是完全免费的，很多插件也都能免费用。

Pro 版本主要是扩展支持力度，比如：

- Clipboard 的云同步、更大的容量。
- 以及主打 AI 能力：可以在聊天框里直接进行 AI 对话。

除此之外，绝大部分内容都可以免费使用。现在也开始支持 Windows 了。

## 使用方式与快捷键

使用方式很简单：

- Mac 上通常是 Option + Space 唤起。
- Windows 上也是 Option + Space。
- 你也可以自定义其他热键、常用快捷键，它都支持。

这个工具是我日常最常用的。在 Mac 上几乎属于装机第一时间会考虑装上的工具，后续使用会更方便。

它更新频率也很高，隔三差五就会更新，比如推出 Raycast Focus 专注模式。每次更新内容都挺多，包括修 bug 和小功能增强。很多插件也会引入新的官方制作、官方定制扩展，帮助你更好地使用系统里的各种软件。像 Raycast Notes 也是最近才推出的比较重要的功能。

它的 Store 页面也做得很好：可以搜索、查看下载量、使用情况，每个扩展都有截图和功能介绍，信息比较详细。
