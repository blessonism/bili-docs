## 虚拟机能解决什么问题

如果你下载了一个神秘软件，却不知道它的安全性；别人借用电脑的时候，担心被他发现一些藏得很深，或者藏得不那么深的小秘密。遇到这种情况，一个好的主意是给电脑安装虚拟机。

虚拟机独立于物理机，可以运行不同的操作系统和应用程序。即使病毒攻击了虚拟机，也不会危及到物理机的安全。如果给虚拟机设置好密码，其他人借用电脑也很难读取到其中的隐私信息。

虚拟机可以很容易地备份、复制和迁移，可以根据需要从某个状态备份或者还原。

## 三款常用虚拟机软件对比：VMware / VirtualBox / Hyper-V

我们来对比一下三款常用的虚拟机软件：VMware、VirtualBox、Hyper-V。

从使用成本来看：  
- VMware：使用成本高，目前官网售价 149 美元。  
- VirtualBox：免费，开源软件。  
- Hyper-V：使用成本低，只需要一个专业版的 Windows 系统即可。

从稳定性来看：  
- VMware：稳定性中。  
- VirtualBox：稳定性低。  
- Hyper-V：稳定性最高，因为它是内嵌在 Windows 系统当中的。

从性能来看：  
- VMware：性能中。  
- VirtualBox：性能最低。  
- Hyper-V：性能最高，启动速度最快，占用的内存最小，支持 GPU 虚拟化技术。

综上所述，我最推荐的虚拟机产品是 Hyper-V。

## 第一步：下载纯净版 Windows 10 安装镜像（ISO）

第一步，获得一个纯净版的 Windows 10 安装镜像。

打开浏览器，在搜索里搜索 Windows10，找到“下载 Windows10”。注意进入的是微软官网，找到“立即下载工具”。

打开刚刚下载好的工具，稍等片刻，点击“接受”。这一步是重点：选择“为另外一台电脑创建安装介质”，点击“下一步”。

这里选择“文件”，点击“下一步”，选择一个保存地址，点击“保存”。耐心等待进度条走完，就可以获得一个纯净版的 Windows10 安装镜像。

## 第二步：确认 Windows 版本是否支持 Hyper-V

第二步，确认 Windows 的版本。

在搜索框搜索“设置”，选择左侧的“系统”，拉到最下面“系统信息”，可以看到 Windows 的版本。

如果此处显示是 Windows 专业版或者旗舰版，就可以使用 Hyper-V；如果显示家庭版或未激活，需要对 Windows 进行升级。

点击“产品密钥和激活”，点击“更改产品密钥”，输入一个专业版或旗舰版的产品密钥，对 Windows 进行升级或激活。产品密钥可以从某宝购买，几块钱就可以。

填好产品密钥，点击“下一页”，系统会自动重启，然后 Windows 就变成专业版或者旗舰版。

如果不想升级 Windows 版本，也可以在家庭版上执行一段脚本，强制开启 Hyper-V 功能。需要脚本的话，可以加关注并私信我，或者从公众号“技术爬爬虾”获取。

## 第三步：开启 Windows 的 Hyper-V 功能

第三步，开启 Windows 的 Hyper-V 功能。

在搜索栏中输入“功能”，点击“启动或关闭 Windows 功能”。在这里选中 “Hyper-V”，点击“确认”。

耐心等待进度条走完，点击“立即重新启动”。等待 Windows 启动后，电脑就拥有了 Hyper-V 的功能。

## 第四步：创建虚拟机并安装 Windows

第四步，创建虚拟机实例。

在搜索栏中搜索“Hyper-V”，点击进入 Hyper-V 管理器。右上角点击“新建” → “虚拟机” → “下一页”。

给虚拟机命个名，比如叫 Win10。如果 C 盘空间比较紧张，可以把虚拟机存放在其他位置。

点击“下一页”指定代数，保留第一代，第一代的虚拟机比较稳定。点击“下一页”分配内存，建议至少分配 4G 以上的内存。

点击“下一页”配置网络，选择 Default Switch。点击“下一页”链接，虚拟硬盘大小随意分配，点击“下一页”。

这里选择“从可启动的 CD/DVD 安装操作系统”，选择“镜像文件”，选中第一步下载的镜像文件，点击“下一页” → “完成”。

耐心等待进度走完，点击“连接”，点击“启动”。就像一台全新的电脑一样，它开始安装操作系统。

安装过程：点击“下一步” → “现在安装” → “我没有产品密钥”，选择 Win10 专业版 → “我接受” → “下一步” → “自定义” → “下一步”。耐心等待进度条走完。

操作系统安装完毕后，虚拟机会重启。这里选择“脱机账户”即可，点击“有限的体验”。

如果使用 Windows11 的 ISO 镜像进行安装，有些细节需要注意，这里不再赘述，可以加关注或进入公众号获取详细教程。
