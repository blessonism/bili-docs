## 这期要讲什么：ToDesk 的 Windows 远程开机

ToDesk 作为一款远程控制软件，在客户端里对未开机的 Windows 设备，可以进行远程开机操作。虽然官方文档对这个功能的介绍已经很全面了，但对一些人来说上手还是有一定难度，所以这期对 ToDesk 的远程开机功能做一个详细介绍，并解答一些常见问题。

## 远程开机的前提：只能 Windows，用网线最稳

远程开机功能只能在 Windows 系统上使用。运行 Windows 的设备可以是台式机、笔记本和一体机，其中台式机兼容性最好、最稳定。

远程开机依赖主板 BIOS 支持 Wake on LAN（网络唤醒）功能。有些主板 BIOS 不支持网络唤醒，就无法实现远程开机。

支持网络唤醒的场景里，网卡分为有线和无线：

- 有线网卡：插上网线，就可以实现远程开机。
- 无线网卡（WiFi）：主板无法直接通过 WiFi 实现网络唤醒，所以用 WiFi 无法实现远程开机。

总结就是：不插网线，就无法使用远程开机；插网线是网络唤醒的前提。

## 台式机/笔记本/一体机分别怎么判断能不能用

### 台式机

台式机只要主板 BIOS 支持网络唤醒，并且用的是有线网卡、插上网线，远程开机一般就没问题，稳定性最好。

### 笔记本

笔记本比较特殊：一般笔记本 BIOS 不包含网络唤醒功能，因为笔记本通常连接 WiFi，而 WiFi 无法直接使用网络唤醒。

也可能有少数笔记本 BIOS 支持网络唤醒。如果笔记本支持网络唤醒，并且网卡支持有线网络，那么：

- 用 WiFi：还是无法远程开机。
- 用有线（插网线）：理论上可以实现远程开机，但这个方法我没有实践过。

如果网卡支持有线网络但机器没有网口，需要扩展坞。使用扩展坞插网线，理论上也可以实现远程开机；不用扩展坞就没办法。BIOS 不支持网络唤醒的话，也无法实现远程开机。

### 一体机

一体机如果主板 BIOS 支持网络唤醒，网卡支持有线网络且有网口，插网线理论上可以实现远程开机；用 WiFi 同样不行。主板 BIOS 不支持网络唤醒的话，也无法实现远程开机。

## 先检查网卡是否支持网络唤醒（Windows 设置网卡参数）

如果网卡本身不支持网络唤醒，那进 BIOS 打开网络唤醒也没用。先在 Windows 里检查并设置网卡：

1. 打开控制面板（桌面没有图标的话，打开文件资源管理器，在地址栏输入“控制面板”回车）。
2. 查看方式改成“大图标”或“小图标”。
3. 打开“设备管理器”。
4. 找到“网络适配器”，选中你的有线网卡，右键“属性”。
5. 在“高级”里：
   - 开启“关机网络唤醒”。
   - 开启“魔术封包唤醒”。
6. 在“电源管理”里：
   - 勾选“允许此设备唤醒计算机”。
7. 点击“确定”。

这样网卡部分就设置好了。

## ToDesk 端必须勾选：开机启动/开机自动启动

ToDesk 的 Windows 端里，打开“高级设置”，把“开机启动 / 开机自动启动”一定要勾上，否则远程开机可能会不成功。

## 让关机很久也能唤醒：路由器做 MAC 绑定（静态 IP 绑定）

官方文档提到：如果想要在设备关机后很长一段时间仍能支持远程开机，需要额外设置路由器的 MAC 地址绑定（静态 IP 绑定）。

这里教一个快速进入路由器网关的方法：

1. 打开控制面板 → “网络与共享中心”。
2. 如果你用的是网线，点击“以太网”。
3. 点击“详细信息/信息”，找到 “IPv4 默认网关”，这就是路由器网关地址（例如 192.168.3.1）。
4. 打开浏览器，输入这个网关地址，登录路由器后台。

以华为路由器为例：

1. 进入“终端管理”，找到这台电脑的 IP 地址和 MAC 地址。
2. 在“更多功能” → “网络设置” → “局域网”里，找到“静态 IP 地址绑定列表”。
3. 点击添加绑定设备，选择你的电脑设备，自动匹配 MAC 和 IP。
4. 确定后完成绑定。

静态 IP 绑定的作用是：在局域网内把这个设备的 IP 地址和 MAC 地址固定住，不再重新分配。

## 进入 BIOS 开启网络唤醒（Wake on LAN）

完成上面操作后，进入 BIOS 把网络唤醒打开。这里给一个从 Windows 快速进入 BIOS 的方法：

1. 打开“设置” → “更新与安全” → “恢复”。
2. 在“高级启动”里点“立即重新启动”。
3. 进入恢复选项后，点“疑难解答”。
4. 选择“UEFI 固件设置”，点“重启”。

重启后会进入 BIOS（UEFI 主板一般支持鼠标操作，这里用键盘示范）。

以映泰主板为例：

1. 在 BIOS 的“高级”选项里找到 “ACPI 设置”。
2. 找到 “PME 从 S5 状态下唤醒”（网卡网络唤醒相关设置）。
3. 把它改成“启用”。
4. 按 F10 保存并重启。

## 家里放一台安卓机做唤醒端：开启 Wake on LAN

可以在家里常驻一台安卓手机：

1. 安卓手机安装 ToDesk 客户端。
2. 登录和电脑端一样的账号。
3. 在安卓端设置里把 “Wake on LAN” 选项开启。
4. 手机插着充电器放家里即可，轻微耗电不用在意。

之后你在日常主力机上就可以用 ToDesk 进行远程开机，开机后再远程控制主机。
