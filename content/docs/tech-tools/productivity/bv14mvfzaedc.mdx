## 告别传统书签：AI 自动整理的收藏工作流

遇到喜欢的网站，这样点一下确实能够收藏，但是很容易吃灰，因为找起来太麻烦了。如果说能够变成这种表格的形式，再利用 AI 自动整理、自动分类，是不是看起来就清晰了很多呢？最重要的是还可以用 AI 进行搜索，不需要记住什么关键词，只需要说出你想要找的网站就行。而且不论是手机还是电脑，都可以很方便地一键收藏，并且统一归纳到这个表格中，然后在任意设备上查看。当然了，某些网站还是建议用书签收藏夹或者是靠脑子去记，会更安全一点。这个东西简单来讲，就是工作流加飞书多维表格。当然，本期视频绝对不是恰饭，纯粹是因为飞书比较适合来干这件事。

## 电脑端配置：安装 Automa 与导入飞书表格

具体要怎么用？我们先来讲讲浏览器上的用法。首先我们需要安装这个叫 Automa 的扩展，它是一个浏览器自动化工具，非常的强大。大家可以打开它的官网去安装，如果无法访问的话，也可以去网盘里获取离线安装包。点击扩展图标，找到这个 Dashboard，打开之后可以先去设置里把语言切换为中文，然后重新打开一下 Dashboard 的主面板就好了。接着去下载这个工作流文件，在新建工作流旁边选择导入工作流，找到刚才下载的文件。这里有个通知权限，建议给一下。然后点击工作流的名称，也就是“收藏链接”这几个字。这里我们可以通过鼠标滚轮来缩小，按住左键来拖动。看起来有点复杂是吧？但是大家只要找到这个“插入数据”，双击一下，然后点击左边的 Insert Data。这里我们需要配置三个参数，放心，很简单的。首先是多维表格的链接，我们可以打开这个表格模板，点击右下角的“使用该模板”，然后就可以在自己的飞书里创建这么一个多维表格。我们可以注意到，左边有“来自电脑”和“来自手机”两个数据表，它们对应的表格链接是不一样的。电脑上需要在选中“来自电脑”后，复制这上面的链接，然后填写到工作流这里就好了。

## 获取飞书应用凭证与权限设置

剩下的这两个参数也很简单。我们打开飞书开放平台，点击创建应用，名称、描述还有图标都随意。这里可以先创建一下版本，然后直接保存并发布。接着来到权限管理，点击开通权限，这边选择多维表格，把后面的权限都勾上，然后确认开通权限。最后来到凭证与基础信息，这里就能看到我们要填写的两个参数了，全都复制粘贴一下。最后别忘了点一下工作流右上角的保存。这样是不是就可以用了呢？其实还差一点点。

## 接入 AI 大模型实现自动分析

我们打开多维表格，点击右上角的三个点，选择更多，添加文档应用，这里添加一下刚才创建好的应用。接着需要点击一下“概述”这一列，选择修改字段列。这里需要填写一下你的 API Key。关于这个之前的视频有详细讲过，这里以硅基流动简单举例。注册登录之后，在 API 密钥这里新建并复制一下密钥，然后填写到这个地方就可以了，很简单对吧？默认会有一些免费额度，当然你可以用这里的免费模型直接替换自定义模型这里的名称就可以了。后面的“详细介绍”那一列也这么设置一下就好了。

## 电脑端一键收藏与视频网站优化

现在真的就可以开始用了。在你想要收藏的页面，点击扩展图标，然后点击这个按钮就会开始处理了，成功的话也会看到类似这样的通知。你还可以在工作流的触发器这里双击，编辑触发器，这里我们可以添加一个键盘快捷键，点击这个按钮就可以录制快捷键。这样一来就可以直接按下快捷键来收藏网页，比点击快捷图标要快很多。处理完之后，在多维表格这里就能看到结果。这几块因为调用了 AI 做分析，所以不会立刻显示结果，处理完之后就是这样的。“详细介绍”这里的内容一般会比较多，所以我们可以配合这个多维表格插件来查看，这样体验也会好很多。如果你觉得这个插件这里的内容太碍眼的话，也可以试试我做的这个。很简单，就只有一个预览功能，但也正因此飞书表示不能全量上架。如果大家想用的话，可以在这里添加自定义插件，填写一下名称，再填写这个地址就可以了。如果你注意一下整个工作流的话，就会发现我对部分视频网站还单独做了优化。除了获取网页文本外，还会尝试获取视频字幕。这样一来，这里的详细介绍就是根据网页文本加视频字幕生成的，会更准确、更丰富。因此如果你乐意的话，也可以自己针对常用的网站做一些优化。

## 利用 AI 智能查询快速找回网页

收藏好之后，最重要的就是搜索查询。注意多维表格的右下角有个“智能记录查询”的功能，这个非常的重要。因为在这里你就可以直接说人话来找链接，用关键词的话就必须得完全匹配上才行，而用这个的话就宽松了很多，也算是用 AI 来做搜索了，有点类似于知识库问答。可惜这个智能查询在手机上还不太好用，只能等飞书优化了。

## 手机端配置：iOS 快捷指令收藏

说完了电脑浏览器，我们再来讲讲手机。iOS 用户直接用这个快捷指令就可以了。注意在这里我们需要设置一下三个参数，记得先清空下原有内容。前两个和电脑上是一样的，最后一个多维表格的链接，我们需要使用这个“来自手机”的表格。因为手机端没法直接获取当前页面内容，所以这里就用了一个可以读取链接的快捷指令来做处理，效果也是很不错的。然后在任意 APP 点击分享，记得使用这个系统分享，找到“收藏链接”点一下就好了。当然你也可以直接复制包含链接的文本，然后手动点击这个快捷指令也可以。

## 手机端配置：安卓 MacroDroid 收藏

安卓这边的话，需要先安装一下 MacroDroid。当然 Tasker 之类的也可以，但是我不会用，所以就选了 MacroDroid。装完之后打开一次，然后把通知权限给一下。它会有一条用来保活的通知，那我们这个流程不需要，所以如果你觉得碍眼，也可以去通知管理里把这个持久通知关掉。然后需要把这个文件下载到手机里，点击它，使用 MacroDroid 打开。这里我们依旧需要设置三个参数，修改方式是直接长按变量，然后点击确定，清空原有内容。前两个和电脑上是一样的，最后一个多维表格的链接，我们需要使用这个“来自手机”的表格。因为手机端没法直接获取当前页面内容，所以这里用了一个可以读取链接的快捷指令来做处理，效果也是很不错的。修改完之后点击左上角的返回，保存一下修改。这时候会提示需要一个悬浮窗的权限，我们点击确定把这个权限给一下，然后一路返回，最后还是点击保存。接着就可以在任意 APP 点击分享，记得使用系统分享，然后再找到这个 MacroDroid 的图标点一下就好了。当然你也可以直接复制包含链接的文本，然后通过长按图标选择这个“收藏链接”也可以处理。或者是在主屏幕这里添加一下这个磁贴，这样就可以复制链接之后，直接在控制中心里点击磁贴来收藏了。但是这两个方式因为要获取剪贴板内容，所以会自动打开 MacroDroid 的本体，因此速度上会有点慢，但也算是挺好用的吧。希望这种收藏链接的方式能够对你有所帮助、有所启发。以上就是本期视频的全部内容了，如果喜欢的话不要忘了点个关注，我们下期再见，拜拜！
