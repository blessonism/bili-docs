## 前言：科研版 Word 设置指南

大家好，我是木木。今天我们来学习一下怎样配置一个科研版本的 Word。因为我的电脑是 Mac，所以今天主要讲的是 Word for Mac 怎样设置。如果你是 Windows 电脑，这个视频仍然是有意义的，你可以根据视频中的引导进行同样的设置，Windows 的设置会更好找一些。

通过这个视频的学习，你可以在打开 Word 的时候就开始写草稿，保存之后就可以直接发给导师。其次，你会对 Word，尤其是 Word for Mac 的基础功能如何设置，包括如何自定义快捷键有一个基本的认识。

视频开始之前，先告诉大家我的设备信息。我的 Word 版本是 2024，如果你的 Word 不是这个版本也没有关系，设置的位置基本上是类似的。我把教程分成了三个部分：第一部分是给老师看的版本如何设置，也就是一些比较基础的设置；第二部分是给编辑看的版本，主要讲复制粘贴到模板中怎样更方便，特别提醒大家，最好不要自己调整格式，粘贴不带格式的文本到模板中是最好的；第三部分是常用的快捷键。

我的教程主要讲究性价比。进行基础设置之后，你可能对一些高级功能还没有什么认识，感兴趣的话后面再去学就可以了。视频里的这些设置首先比较常用，其次设置完之后在所有的文档中基本上都可以用到。

## 核心设置：自动保存与备份

我现在打开了一个 Word 文档，首先进行一个最重要的设置：自动保存。我用的是 Command + 逗号，打开 Word 的偏好设置（后续说的首选项或设置都是这个面板）。

在这里点开“保存”，勾选“保存自动恢复信息”，然后在“保存间隔”这里选到最短的，也就是1分钟保存为 Word 文档就可以了。如果你还是不放心，可以勾选“始终创建备份副本”。如果你还不放心，可以把其他的选项也勾选上。

提醒大家一下，如果更改了设置，最好重启软件，这样能确保设置生效。同时建议大家把 Command + S 或 Ctrl + S 焊在手上，没事就保存一下。

## 视图设置：显示非打印字符

第二个设置是显示非打印符号，这主要是方便我们校对。在设置界面点击“视图”，然后把“显示非打印字符”的“全部”选上。

我打了一个中文的空格和一个英文的空格，你可以看到在这里它都显示出来了。这些符号如果不进行刚刚的设置是不会显示出来的，你可能就无法分辨字间距是因为对齐导致的，还是因为有空格导致的。

## 功能区自定义：查看脚注分隔符

下一个是在功能区固定一个“查看脚注分隔符”。在“功能区和工具栏”点开它，选择“不在功能区中的命令”，这个列表是按照拼音顺序排序的。我们在里面找到“查看脚注分隔符”。

脚注一般是在“引用”这个地方，我们在“引用”这里新建一个组，然后选择大于号把它放进去就可以了。点击保存之后，你可以看到在“引用”标签下出现了一个新的功能，就是“查看脚注分隔符”。

我们来插入一个脚注看一看。在这里可以看到，脚注分隔符前面缩进了两个字符。如果我们不想这样，就点击“查看脚注分隔符”，然后进行格式设置，把“首行缩进2字符”取消掉就可以了。如果你不想让它显示在这儿，直接把它往下拖，它就消失了。这个设置主要是因为有时候碰见标题需要写注释，改分隔线会稍微方便一点。

## 样式修改：正文字体与段落对齐

很多用 Mac 的同学把文档发给别人之后，对方显示不对，很崩溃。其实大部分是因为自己没设置，因为 Mac 有一个字体叫“宋体-简”，但那个字体其实是不对的，我们应该用“宋体”。

点开“样式”，找到“正文”，把鼠标放上去，右键选择“修改”。这里的名称不要改。“后续段落样式”的意思是，我们在正文的下一段如果回车，下一段基本上也是正文。如果你改的是标题格式，后续段落样式也最好改成正文。

在这里我们主要改两个设置。一个是字体设置：中文是宋体，西文是新罗马（Times New Roman）。字形和字号选常规、小四号（一般情况下的论文是这样的，或者你导师有别的要求也可以放大一点），然后点确定。

下一个是段落对齐方式。这里选“两端对齐”，当然如果你导师喜欢左对齐，那就左对齐。“大纲级别”的意思是它在目录里面显示的是几级，正文肯定不会出现在目录里，所以大纲级别就是“正文文本”。如果你设置大标题，就应该是一级，接下来是二级、三级标题。

在这个地方要注意选对缩进，首行缩进2个字符。有些同学可能说，我在这输入了一个2，点击确定之后它变成了2磅，不是2个字符，怎么办呢？你就直接在这手动输入“2字符”就可以了。

我在这里把所有的对勾全都取消掉了，段前段后我都写的是0行，行距选的是单倍行距，然后点确定。设置完之后应该跟我预览的是一样的。

在这里你要勾选“添加到模板”和“添加到快速样式列表”，这样正文的样式就会更新到我们刚刚的状态。

## 字体安装：补充 Mac 缺失的常用中文字体

这次我找到了一个网站，感谢浙江师范大学外国语学院提供的字体文件。把字体文件下载下来解压，就可以找到一个文件夹，里面就是我们平常要用到的字体。

我不建议大家都装，需要哪个装哪个就好。最重要的是把“宋体”装好，还有“方正小标宋简体”，这是我们一般情况下标题的指定字体。如果你不太确定自己的字体是不是全的，全装也行。

双击“宋体”，它会跳转到验证界面，你可以看到这个宋体长什么样，它的名字叫“新宋体”。如果你没有的话点击安装就好。安装完之后，就可以在“我的字体”里发现宋体了。

对于一些不太需要的字体，你可以右键点击“取消激活”，它就会变成灰色的样式，不会出现在列表里了，避免混淆。如果像“宋体-简”有小锁不能取消激活，那也没有关系，不要选它就好了。

## 图片排版：利用表格对齐多张图片

好多同学在排多张图片的时候不知道怎么办。你有两个选项：第一个是放在 PPT 里排，好处是可以随便调整位置，也可以抠图，但是复制来复制去可能会导致图片字体大小不对。

如果是横平竖直的排列，直接插入表格就可以。我们插入一个两列的表格，插入两张图片试试。图片太大了我们缩小一下。如果是你自己用 Python 或 R 画的图，最好在画图的时候就选好字体大小，插入后不要轻易去缩放改动，否则字体大小就变了。

在表格里排版的好处是，你可以输入小序号，也可以在“表格布局”里进行对齐调整，或者增加两列进行调整。

有些同学说，我这个宽度正好跟 Word 版面的宽度一样大，但是复制粘贴进来之后发现它有边距。怎么办呢？点开“表格属性”，在“单元格”这里点开“选项”，把左右的单元格边距都调整为0就可以了。调整完之后，在“表格设计”里把边框全都去掉。我自己一般的习惯是不会把字和图片排到一个格里的，我觉得那样很难受，但你也可以这么干。

## 冷知识：直接复制 PDF 为图片

告诉大家一个冷知识：Mac 是可以直接复制 PDF 粘贴进 Word 的，这个比 Windows 要方便很多。你把 PDF 复制粘贴进来，直接就是图片。

## 插入与修改题注格式

之前已经教过大家怎么去引用了，这里只教大家怎么插入题注。在“引用”里点击“插入题注”。标签有好几个，如果没有你想要的（比如 Figure 或 Table），点击“新建标签”就可以。

我们现在用“图”这个标签。编号格式可以是 1、2、3，也可以包含章节号，写大论文的时候包含章节号会稍微方便一些。点击确定，它就出现了“图1”。

我们管它叫字体设置。你可以发现这里的格式好像是错的，字体是黑体，不是我们正文的样式。为什么呢？我们在样式窗格找到“题注”，右键修改。其实黑体是一个比较常见的字体，如果你的论文要求不是这样，给它改了就行。我一般是全文都设置新罗马加宋体的混排，跟刚刚正文的设置是一样的。

## 图像设置：防止 Word 压缩图片质量

保存的时候可能会遇到一个问题，就是 Word 会自己压缩图片质量。打开首选项，在“编辑”这里有一个“图像大小和质量”。你可以选中“所有的新文档”，勾选“不压缩文件中的图像”，然后把“默认分辨率”选到“高保真”，保存就可以了。

但是我比较缺乏对这个世界的信任，所以我每个新文件都会重新设置一下。相对来说比较麻烦，所以我一般是把文件里所有的图片打包发给老师，每一张图片以它在正文里的题注文字来命名。

## 快捷键自定义：默认无格式粘贴

提交审核的版本，怎样直接复制粘贴进模板比较方便？最方便的就是复制之后，粘贴时不带字体格式，直接匹配正文格式。

怎么做呢？在 Mac 屏幕最顶部的菜单栏点击“工具”，选择“自定义键盘”。点击“编辑”，这里有好多命令，我们主要改两个。

第一个是 `Edit Paste`（编辑粘贴）。这里原先可能是 Command + V，你需要把这个删除，定一个新的快捷键，我用的是 Ctrl + V。点击“指定”，然后点确定。

第二个是 `Paste Text Only`（仅保留文本粘贴）。我们给它分配快捷键 Command + V。因为 Mac 的 Command 键通常对应 Windows 的 Ctrl 键，所以我们把这个改成最常使用的快捷键。

注意：如果你先改了 `Paste Text Only`，后改 `Edit Paste`，快捷键就会冲突，后面的快捷键会被挤掉。所以顺序很重要，一定要先改 `Edit Paste`，再改 `Paste Text Only`。设置完如果没生效，就去重启一下软件。

## 隐私设置：保存时移除个人信息

保存的时候有一个匿名的问题。点开“工具”，选择“保护文档”，点击“隐私”，勾选“保存时从文件中删除个人信息”，点击确定。

这样你按 Ctrl + S 保存之后，下次打开它，在文件属性里可以看到摘要里的作者信息消失了。如果不勾选这个设置，文件的作者就会是你设置的作者名。

## 常用快捷键与备份建议

最后一部分是常用的快捷键，我列了几个：

第一个就是 Ctrl + S 或 Command + S。这个键一定要焊在手上，没事就保存。但也提醒大家，Ctrl + Z / Command + Z 撤销回滚是有次数限制的。所以最好的方法是，你沟通的版本或自己写的版本，要保留多个文件。最好有所有的历史记录，或者如果你了解 Git，可以通过 Git 来做版本控制。我自己觉得最有安全感的方式就是存好多个文件，每一次的历史记录我都能记得。

第二个是复制粘贴。Word for Mac 的复制粘贴快捷键是有延迟的，所以最好每次复制的时候多按几次，才能粘贴过来。如果 Command + C 不管用，就用 Ctrl + C 试试。

第三个是段落格式。这个对我来说比较重要，它的快捷键比较特殊，是 Option + Command + M。我想不到它跟段落格式有什么关系，所以特意记了一下。

如果是字体设置，快捷键是 Command + D。这个面板跳出来会比较慢，需要的话可以记一下。

有空可以学一下 Word 怎么替换。Word 的替换是非常厉害的功能，深度使用会发现非常好用。不想学也可以不学，等需要的时候去搜一下就可以。
