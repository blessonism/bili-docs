## 开场与本期主题

Hello，观众老爷大家好，我是大和李斯特。欢迎来到看番教程的第五期。

今天带大家进入播放器的实际应用：视频播放器需要怎么设置，才能达到最佳画质。

## 播放器怎么选：为什么用 PotPlayer 讲

PC 端主流播放器差不多是 PotPlayer、MPC（MPC-HC / MPC-BE）和 MPV，这些各有利弊。

本期主要围绕 PotPlayer 展开，但不是说 PotPlayer 一定比其他播放器好用。只是从上期投票比例来看，PotPlayer 的使用率仍然相当高。

相比 MPC 和 MPV，PotPlayer 最大的优势是 UI 整洁美观、交互简单直接，并且网络播放功能很强，可以说是最容易入门上手折腾的播放器之一。

但它最大的问题是预设方案非常糟糕，啥也不调基本没法看，需要大量修改才能勉强和其他播放器站到同一起跑线。不过从教程角度来说，一边调整设置一边讲解，反而更容易让大家理解播放器的构成，而且大多知识和其他播放器也相通。

等后续内容讲完之后，我也会再对其他播放器补充方案。有需要的可以先点个关注。

## 准备工作：安装包下载与注意事项

首先准备 PotPlayer 安装包和 SVP 安装包。

先别急着说不想要补帧，后面你就知道为什么要下这个了。这两个在各自官网都能搞到。

SVP 不需要补帧的话可以不用买，只要下载安装包就行。SVP 直接百度搜索就能找到官网。

要注意 PotPlayer 的官网并不能在百度中搜到，建议直接用域名访问，我也会在简介里贴出地址。如果没有梯子，截至视频发布时，最新的 PotPlayer 安装包我也会放在百度网盘，链接在简介里，有需要自取。

## 安装 SVP：只拿 madVR 和 LAV（不补帧也建议装）

我们先装 SVP。打开安装后，基本没啥好说的，直接下一步，安装路径推荐默认别改。

这里要注意：先别急着下一步，选择 “Switch to the component selection”。展开组件列表后，才是我们真正需要的东西。

如果你完全不想要补帧，只需要勾下面两个：

- madVR  
- LAV Filters

其他都可以不装。

madVR 是渲染器，LAV 是一套解码器和分离器。这俩自己找资源装其实很麻烦，官方资源不好找，版本也很乱，很多教程资源包下下来，可能电脑里装出多个版本的 madVR 和 LAV，头都大。

SVP 的好处是能确保装到最新、干净的官方版本，同时用 SVP Manager 也方便管理与更新。所以就算你不用 SVP 的本体，也建议装一个。

如果你不反感补帧、也想试试，那除了上面两个插件，还需要再勾图里的其他东西（作用我也标了）。补帧会在之后的篇章专门展开讲。

然后一路下一步，等待下载安装就行。最后如果你不用 SVP 本体，完成后直接退出，别让它运行；要补帧的话你可以先开着。本期先不讲补帧。

## 安装 PotPlayer：安装选项怎么选

接下来安装 PotPlayer。到第一次有东西可调的地方开始讲：

这些选项都挺直观，按自己喜好选就行。

- 如果你电脑上就这一个播放器，可以关联所有格式。
- 安装位置建议默认，因为不少插件会和路径有关；全部默认是最省力、最无脑的做法。

到完成页面时注意“额外编解码器”：我们会自己准备，不用装它默认的垃圾，所以别勾。

“Detect HW decoder/encoder” 勾上。

安装完运行 PotPlayer，但先别急着放片，这玩意要调的东西不少。

## PotPlayer 设置：关掉内置处理，改用 LAV 全家桶

右键播放器区域 → 选项 → 滤镜。

先到“内置图像处理器”：稍微不启用，下面的勾全部去掉。PotPlayer 内置的基本没啥好东西。

然后到“源滤镜/分离器”这边，点开“滤镜/解码器管理”。我们要从系统滤镜里，把前面 SVP 装进系统的 LAV 全家桶都塞进去。SVP 安装过程中会把这些注册成系统滤镜，方便就在这。

这里解释一下：

- **LAV Splitter**：只是分离器  
- **LAV Splitter Source**：除了分离器，还包含读取文件的 File Source（源滤镜）

仅分离器的话，读取文件的工作会交给 Windows。两者最后效果差别不大，但建议用 **LAV Splitter Source**。

剩下的：

- Audio Decoder：音频解码器  
- Video Decoder：视频解码器

添加完之后，每一个都选中一次，把右边能打勾的格式全打勾，确保 LAV 支持的文件都走 LAV。完成后点确定。

回到之前的滤镜界面，你会发现分离器选项里能选 LAV 了。把所有能换成 LAV 的全部换成 LAV，不能换的就保留默认推荐。

一个小技巧：LAV 的选项一般都在可选内容的最后一项，你不用每次点开设置，直接鼠标滚轮往下滚，能很快完成设置。

视频解码器和音频解码器面板同理，照做就行。

## 切换渲染器：启用 madVR

来到“视频”选项页，把“视频渲染器”换成 **madVR 视频渲染器**。这样渲染器就挂上了。

这里必须先在 SVP 那边装了 madVR，不然你在 PotPlayer 里找不到。

到这里为止，PotPlayer 画面相关的核心调整就结束了。

## 个人偏好设置：缩略图、全屏独占、补帧滤镜

接下来是一些个人喜好设置：

- 播放设置页里，有“鼠标指向显示缩略图”功能，挺好用，方便按预览画面准确拖进度条。
- 视频设置页里，有是否开启“全屏独占模式”的设置，按需求选择。类似游戏里的无边框全屏和真全屏的区别。

如果你需要 SVP 补帧：在“滤镜”→“全局滤镜优先权”里添加系统滤镜 **Avisynth Filter**，并设置为强制使用。

## 显卡设置：主要检查输出参数

显卡设置这块，如果是比较新的硬件和驱动，现在基本不太需要额外调整，默认通常已经是最优。

主要检查两点：

- 显示器是否为最高颜色深度输出  
- 输出动态范围是否为“完全”

不同厂家显卡设置位置略有不同，但都差不太多。

## 播放检查：确认 LAV 与 madVR 是否生效

终于可以开片检查了。随便找个片子，用 PotPlayer 打开，按一下 Tab，可以看到详细播放信息。

重点检查：

- 视频解码器、音频解码器是否都是 **LAV**  
- 视频渲染器是否为 **madVR**（或 madVR video renderer）

如果都没问题，就可以开始享受高清看片体验了。当然，过不了多久你就会发现，这只是折腾的开始。
