## 免费刷绿墙工具介绍与下载

GitHub贡献墙空空怎么办？可以去闲鱼买，但是今天主播教大家免费刷绿墙。在GitHub搜 zm-rf-t GreenWall，这是主包写的开源工具“绿墙”，可以不提交记录，刷各种图案。我们点击Release即可下载exe执行文件。软件开源，报毒正常，放心食用。

## 软件设置与自定义画图

打开软件主界面，下面就是刷绿墙的区域。找到你的GitHub用户名，在设置里找到你的GitHub邮箱，填写到软件里，仓库名可以随便写。这里可以滚动鼠标滚轮选择年份，黑色是未来的日期，不可贡献。右下角切换中文，鼠标悬浮格子上面会显示日期和贡献次数，拖动鼠标画画。主包换到2024年，每次只能生成一年份贡献。好了，主播画好了，还可以用橡皮擦擦掉。

## 生成本地仓库并推送到GitHub

点击生成仓库。生成有点慢，可能会等3分钟，成功后自动跳出仓库文件夹。我们打开CMD，准备把本地仓库推送上去。但是在这之前，我们要新建一个仓库，名字随便写，可以跟刚才的仓库名一样，其他设置千万不要动，点生成就OK。好啦，我们把地址复制下面的命令。主包这里用SSH，大家用HTTPS也可以，用 `git push origin master`。Git命令不会的问AI或者看仓库的README。

## 刷新主页查看贡献墙效果

回到仓库刷新一下，看看推送成功了没有。推送上来了，我们回到个人主页，可以看到已经有绿色了。一般等个一分钟到两天，GitHub才会完全显示你的贡献。主包运气比较好，立马显示出来了，可以看看一不一样。
