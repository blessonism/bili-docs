## 开源项目 ICS：一个项目让你掌握 CSS

今天分享一个 GitHub 神级开源项目 ICS。这个项目可以说包含了市面上你能见到的所有 CSS 效果。

## 纯 CSS 动画案例：拓宽思路与细节

项目里有一些天马行空的纯 CSS 实现的动画效果，不仅能拓宽你解决问题的思路，还会涉及一些你容易忽视、或者很有趣的 CSS 细节。有趣的话题加上天马行空的内容，对 CSS 初学者来说相当友好，可以让你快速从新手成长为 CSS 大神。

## 由浅入深的讲解方式：从基础到实战

我们点进去看看它是怎么实现的。每个 CSS 效果作者都会由浅入深地进行讲解，先从这个动画要具备的 CSS 基础知识讲起，比如 CSS 变量、style 属性、keyframes，再结合实战演练，最终一步步实现开头的 CSS 效果。

如果你是编程初学者，通过这种由浅入深的讲解方式，相信也能很好地掌握一些 CSS 知识。
