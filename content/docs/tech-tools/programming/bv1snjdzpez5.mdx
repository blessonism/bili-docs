## GitHub Star数最高的开源项目

这是GitHub目前为止star数最高的项目，叫做FreeCodeCamp，已经拿到了419k的star。它是一个开源的、非盈利性的主打实践的教学性质项目，有点类似于之前这一期讲的Developer Roadmap。这个项目由5000+的开发者共同维护。Developer Roadmap是一个十分全面的学习路线项目，FreeCodeCamp则是一个更加注重实践的学习项目。它的目的是帮助更多的人转型进入科技行业，所以这个项目更加注重的是实践能力。

## 面向开发者的英语课程与求职准备

我们进入到项目的网站内部，从上到下看看这个项目包含的内容。首先是面向开发者的英语课程，这里提供了大量的开发者相关的英文课程。比如我们点开第一个，学习第一天上班的问候语，选择第一个课程，就可以通过听、写去学习开发者相关的英语。这里有大量的学习案例，我还蛮心动的。英语学习之后，第二块是求职准备。

## 渐进式的全栈开发与代码学习

第三块就是前后端全栈，加上Python数据分析以及机器学习的部分了。代码学习部分都可以直接在网站上进行代码编写，这里都是以实际的项目出发让学习者进行学习。比如第一个课程就是使用HTML和CSS来完成整体的项目，并且是渐进式的学习路线。前端的技术还是比较全面的，这里的关系型数据库用的是PostgreSQL，后端开发语言用的是Node.js以及Express框架，倒是有点像从前端的角度去做全栈开发会用到的技术栈。再往下就是Python数据分析、机器学习相关的内容，这些就是这个项目的全部组成部分了。

## 适合初学者与转型者的引路项目

通过FreeCodeCamp可以让学习者完全熟悉全栈的开发方式，之后就算换到不同的语言都可以快速上手。从我个人的感觉来看，这个开源项目更加适合代码初学者，或者要转型到程序员这一行的人，这个满是实践的项目会是个很好的引路项目。我自己的话，对开头这个英语学习部分还挺感兴趣的，大家有兴趣的可以去体验体验。以上就是本期视频的全部内容，我是于仔，我们下期再见。
