## 一键生成代码注释

来给你们看一下我写的 VS Code 扩展有哪些好用的功能。我们在阅读一些代码的时候，总是会抱怨别人为什么不加注释，我压根看不懂，特别是那些屎山代码。这个时候我们就可以点击这个 icon，一键加上注释，方便阅读。没错，它仅仅是方便你阅读，不会修改原文件。这是一个临时文件，不会提交到 Git。当然你也可以 copy 到这里来，把注释加上也可以。它仅仅是阅读辅助，但是非常高效，非常有利于你阅读屎山代码。

## 跨语言逻辑转换

在我们学习一门语言，或者打开一个不怎么熟悉的语言的项目时，比如我是 Python 开发者，我看到这个 JS 文件，这什么鬼啊，我压根看不懂，能不能转化一下？能的，一键转成 Python 方便我阅读。仅仅是方便阅读，它不会执行，但是有利于我立马了解其中的逻辑。对照着看，我一眼就看出来原来是这样子，那我改起来就得心应手了。

## 批量复制文件为提示词

以前要一个个文件复制到浏览器去问 AI，太麻烦了。能不能一次性复制好几个文件？能。Copy 直接粘贴为提示词，好几个文件就作为提示词发给 AI，多方便。而且它可以复制整个文件夹，或者文件夹加文件，非常的好玩。

## 执行自定义指令与开源获取

它还能选中文件，执行自定义指令。我这里自定义用了第三方的一个 Aider 去执行，在命令行工具询问这些文件，非常的 nice。这是一个非常高效率的工具，你可以在 VS Code 上面搜到，这是免费开源的 AIDE。我把它放到 GitHub 上面了，喜欢的可以点一下 star，免费开源 MIT 协议。
