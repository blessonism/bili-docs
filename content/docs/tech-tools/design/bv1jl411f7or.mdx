我真的是要被自己蠢哭了
就是这个传统的回形纹路
以前我都是用钢笔工具画出一段
然后自己拼了半天才完成
而且后续要改成其他的形状也非常的麻烦
我到今天才想起来
我可以直接拖入画笔面板中建立图案画笔
然后就可以在任意路径中使用
点击这个对象选项
把大小调整到合适适当
加上一点点的间距
在执行对象扩展外观之后呢
就可以随意调整粗细了
而且要更换其他的形状也是轻而易举
其他的图案也可以用同样的方法
简直是一劳永逸呀
