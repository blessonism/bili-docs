## 课程目标：模仿何同学的颜文字动画效果

今天要做一个效果，是根据何同学的视频来拆解思路做的。平时大家看综艺、电影电视剧、刷抖音或者短视频，看到特效包装时，慢慢会形成一种习惯：会去想这个东西到底怎么做出来的。

有的效果看起来很简单，做起来工序很复杂；也有的看起来很难，做起来反而很简单。这个过程需要不断思考、摸索，多看别人的作品来学习，提升自己的思维方式。

今天先做第一个部分：颜文字动画，也就是平板里显示的内容。这部分不是后期跟踪做出来的，而是先把平板内容做好再拍摄。我们会做 4～5 个颜文字的变化效果。

## 准备颜文字素材（PS → AE）

先找到颜文字。颜文字是在 PS 里先做好的：

1. 网上搜索颜文字内容，挑自己喜欢的。
2. 复制到 PS 里。
3. 有的表情符号不显示，就在文字工具里换一个字体，哪个能显示就用哪个。
4. 存成 PSD（比如保存到桌面）。
5. 把 PSD 直接拖进 AE 使用。

## AE 新建合成并拆分 PSD 图层

导入 PSD 后新建合成，把颜文字 PSD 拖进合成。

成片里这个文字会抖动，而且是一个个颜文字在变化：有时脸动，有时手动。但 PSD 里目前是整体，所以要先拆分。

做法是把同一个颜文字图层复制多份（Ctrl + D），每一份只保留一个部件：

- 比如第一个颜文字分成 7 个部分，就 Ctrl + D 复制 7 份。
- 然后逐份裁出对应部件，并重命名（左手、左脸、眼睛、嘴巴等）。

这么做的原因是：最终图层会非常多，编号可能到 40 多层。前期养成重命名习惯，后期操作会轻松很多。

## 第一个颜文字：关键帧动画（出现、遮罩、透明度）

### 脸的出现动画（位置 + 不透明度）

先做脸的动态：

- 例如在第 15 帧作为最终出现的位置。
- 往前挪到开始帧，让它从外面移动到中间。
- 再加不透明度（T），让它从 0 到 100 出现。

### 眼睛动画（蒙版路径）

眼睛用蒙版来做：

- 在结尾处打蒙版路径关键帧。
- 回到开头改变蒙版形状，把眼睛遮住。
- 调整到完全遮住为止。

### 嘴巴与手的动画

- 嘴巴：用不透明度切换（到某帧从 0 → 100）。
- 左手：用不透明度。
- 右手：用蒙版。

这样基础变化就完成了。

## 让颜文字持续抖动：空对象 + wiggle 表达式

成片里表情包还会一直晃动。最常用的是旋转摆动，这里用表达式更方便。

1. 新建空对象。
2. 把空对象锚点放在正中间，左右摆动才均匀。
3. 选中空对象旋转（R），按住 Alt 点击秒表，输入表达式：

   `wiggle(频率, 振幅)`

   - 频率：一秒抖动多少下，比如 5。
   - 振幅：旋转幅度，比如 10（觉得太明显就改 5）。

4. 让所有拆分出来的部件图层，父级指向这个空对象（Parent 到空对象）。

这样所有部件就会跟着空对象一起抖动。

## 第二个颜文字：复用流程并做更自然的过渡

同理做第二个颜文字：

1. 放入第二个颜文字。
2. 看它由多少部分组成（比如 9 个），Ctrl + D 复制 9 份并拆分。
3. 按第一个的节奏调整过渡更自然：比如第一个在 15 帧完成，那第二个可以从 30 帧开始做。

### 示例：脸、手、小花、嘴的关键帧安排

- 脸（左右脸）：位置（P）做 10 帧左右的展开。
- 左手：位置（P）+ 缩放（S），锚点先摆好，做一个缩放推进的动作。
- 右手：位置（P）+ 旋转（R），锚点放在手的底部，做甩动。
- 小花、嘴巴：直接用不透明度（T）在 30～40 帧从 0 到 100。

### 删除不需要的前后段

不需要显示的部分用工作区裁剪掉：

- 选中图层，Alt + [ 裁掉前半段。
- Alt + ] 裁掉后半段。

### 给第二个也加抖动

- 新建空对象（或复用同类做法）。
- 让第二个颜文字拆分层父级到空对象。
- 把它裁到只保留需要的时间段。

## 第三个及更多颜文字：隐藏图层降低干扰

继续按同样逻辑做第三个、第四个、第五个。图层会越来越多，看起来很累，可以用图层开关把不需要操作的部分隐藏（仅隐藏列表显示，不影响预览窗口效果）。

第三个颜文字示例：

- 拆分（比如 6 个部分），逐个重命名。
- 关闭原始未拆分的那层“小眼睛”（原始整体层），避免干扰。
- 重点调整嘴巴：位置（P）+ 旋转（R），锚点放正中间。
- 汗的部分：用不透明度（T）从 0 到 100，关键帧稍微错开一帧更自然。
- 最后把拆分层父级到空对象，保持抖动一致。

## 背景颜色变化：纯色层叠加 + 不透明度

最终效果里还有背景变化：

1. 新建纯色层，先做一个蓝色背景。
2. 合成如果之前是 80×80，改成 1920×1080，背景也会跟合成一致。
3. 再加一个不同颜色的纯色层。
4. 通过调整其中一个背景的不透明度（T），让它在某个时间点开始变色。

这个内容做好后需要导出，放到平板上，后期用单反拍摄。

## 跟踪部分：素材选择与预处理（录课 2）

这节接着做跟踪。素材是前期拍好的，最终效果会用到：

- 颜文字动画叠加到画面中。
- 摄像机跟踪（让颜文字贴在画面里的某个点上）。

### 选素材的注意点

尽量选明暗分明、画面清晰、有动感的素材。不然 AE 有时解析不出来。

前期拍摄时就要想到后期要跟踪，所以尽量找明暗分明的区域当记号点。

## 截取片段与时间码归零

随便截一段：

- 选开始位置按 B，结束位置按 N。
- 右键：将合成修剪至工作区域。
- 为了方便后续操作，Ctrl + K 把开始时间码调到 0:00。

## 先稳像再跟踪：变形稳定器 + 预合成

播放会发现有抖动、不够匀速，先做稳定：

1. 给视频加“变形稳定器”，让画面更稳定。
2. 稳定后不要直接加“3D 摄像机跟踪”，有时会报错。
3. 先预合成：Ctrl + Shift + C。
4. 在预合成后的那层上添加“3D 摄像机跟踪”。
5. 跟踪点可以调大一点，方便看。

## 找稳定跟踪点

找一个从头到尾都没消失的点作为跟踪点。确认这个点一直存在，就用它。

## 制作要跟踪的颜文字动画（新的一套）

把颜文字拖进来，流程和前面类似：

1. 拆分：比如 8 个部件，就 Ctrl + D 复制 8 份并拆解。
2. 关掉原始整体层的小眼睛，避免干扰。
3. 做动画（示例）：
   - 眼睛：到第 20 帧打位置关键帧（P），开头把它放到中间；再加不透明度（T）让它从 0 慢慢出现。
   - 两个小等号：同样做位置动画，让它们开头藏在脸两侧。
4. 因为图层已经做过蒙版，再在原始层上继续做蒙版容易混乱，所以先预合成（Ctrl + Shift + C），再继续做后续动画。

### 加一点立体感（3D 开关 + X 轴旋转）

让它从“平躺”到“立起来”：

- 给右手、右脸、嘴巴、左脸等层打开 3D 开关。
- 通过 X 轴旋转做一个从躺平到站立的过程。

### 抖动依旧用空对象 + wiggle

- 新建空对象。
- 空对象旋转加 wiggle 表达式。
- 把拆分出来的图层父级到空对象。
- 注意不要把最下面那层原始整体颜文字也一起父级进去（那层一般是关掉的）。

## 让颜文字贴到画面上：绑定到跟踪空对象

跟踪有两种常见方式：替换法、复制空对象位置信息。这里用“父子关系绑定”的思路。

1. 使用 3D 摄像机跟踪的跟踪点创建空对象（用于跟踪）。
2. 颜文字图层也要开 3D 开关，这样它有 X/Y/Z 三个轴。
3. 建立父子关系：把颜文字父级到跟踪空对象。
4. 由于 3D 下移动可能要带 Z 轴，拖动调整时按住 Shift，避免只在二维里跑。
5. 调整颜文字大小与位置，让它贴合画面。

## 做一个跟踪圆环（空心圆）并与跟踪绑定

做一个 100×100 左右的空心圆：

1. Ctrl + Y 新建纯色层。
2. 用矩形工具画蒙版，做出空心圆效果：让蒙版 2 对蒙版 1 做“相减”（Subtract）。
3. 做“从无到有”的出现动画：
   - 在结束位置（比如 10 帧）打“蒙版扩展”关键帧。
   - 回到开头把参数调小，让它慢慢扩展出来。

让圆环也跟踪并有空间角度：

- 给圆环层开 3D 开关，父级到跟踪空对象（按住 Shift 绑定更稳）。
- 调整大小。
- 圆环默认面向镜头，让它稍微“躺下去一点”，调它的旋转值（按住 Ctrl 可吸附数值）。

最后做圆环和线条的出现节奏：

- 圆环先出来：缩放（S）开头为 0，逐渐到正常。
- 圆环出来后，颜文字再出现。
- 预览检查：圆环、线条、颜文字跟随画面抖动的过程完成。
