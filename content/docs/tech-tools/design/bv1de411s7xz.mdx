## 把文字打进 PS，先选对字体
先把文字打到 PS 里，替换成合适的字体。一般都是宋体。

## 套索选背景，用内容识别把原字消掉
用套索框选背景文字区域，然后用内容识别把字消除。别去调透视，那样很二百五。

## 自由变换配合 Ctrl 调整位置形变
按 `Ctrl + T`，按住 `Ctrl` 去调，让文字跟原图的角度、形变对上。

## 用滤镜做边缘过渡：高斯模糊 + 风格化扩散
在滤镜里先用高斯模糊，再用风格化里的扩散，把边缘的溢出和那种“假滑”的感觉压下去，瞬间就有那味儿了。

## 加杂色 + 混合选项压进底图，完成替换
滤镜里添加杂色。

双击图层进混合选项，用“下一图层”的白色滑块去融合（按住 `Alt` 向左拖分开滑块），文字就替换好了。
