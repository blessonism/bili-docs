## 60秒复刻黑洞海报

昨天有个同学问我，这张被甲方一稿过的海报是怎么做的。来，60秒搞定它。

先画一条直线，执行「效果 → 扭曲和变换 → 变换」，调整角度，副本数拉高，做出一圈线条。

再画一个圆，执行「效果 → 扭曲和变换 → 粗糙」，调整大小和细节。选中形状后，点击「对象 → 扩展外观」。

使用形状生成工具，减去中间的部分。再用旋转扭曲工具、褶皱工具调整整体效果。

画一个圆，填充黑白渐变，全选后添加蒙版。复制一层，放大并旋转角度，重复上一步操作两次完成。
