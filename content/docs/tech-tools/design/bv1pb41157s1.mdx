## 开场与效果预览

Hello，大家好，欢迎收看本期的视频。这节课讲解 PR 实现水墨转场效果。先看一下效果，我们开始讲如何在 PR 中实现这种水墨转场。

## 素材准备

在 PR 中准备三条视频素材，还要准备一些水墨转场的视频素材。我这里用的是 006，其中有一个在视频的最后会分享给大家。我也会把这些水墨转场视频素材分享给大家。

## 基础操作：摆放轨道与裁切水墨素材

详细看一下具体操作。

把视频 1 放在 V2 轨道上，稍后会知道为什么要放在 V2。然后拖动一个水墨转场素材到 V3 轨道上，播放看一下。

到达这里的时候已经全部变成黑色，后面就不要了。按快捷键 C 调出切刀工具，把它切断并删掉，再把后面进行对齐。

## 给视频 1 添加「轨道遮罩键」并设置亮度遮罩

我们需要给视频 1 添加一个效果。在效果面板中搜索“轨道遮罩键”，拖到视频 1 这个素材上。

回到效果控件里，遮罩选择“视频 3”，也就是 V3 上的 006 素材。如果素材在四轨道上，这里就选视频 4。因为现在放在三轨道上，所以选择视频 3，也就是拾取视频 3 这个素材的黑白信息，让视频 1 进行显示与不显示：白色的地方显示，黑色的地方不显示，变成透明。

选择视频 3 后会发现没有任何东西，因为要拾取它的亮度信息，所以选择“亮度遮罩”。这时就可以看到：刚开始白色的时候依然显示原来的素材；当水墨素材是黑色的地方，这里其实不是黑色，而是透明。

把视频 2 放在下面，就可以透出视频 2 这个素材，这就是基本原理：通过拾取黑白信息，让视频素材变成透明。播放看一下。

## 视频 2 的转场：调整水墨素材与加速

视频 2 下面也要用到视频 3，所以还是要把这个水墨素材切断，把它往上提一下，然后把视频 3 挪到它的下面。

再选择一个水墨素材，比如 001，播放看一下，拖拽过来。音频不要删掉。

大家可以看到水墨的演变时间有点长，我们可以对素材进行加速：按快捷键 RR，然后拖动进行加速到这里。再播放看一下。

同样，对视频 2 添加一个“轨道遮罩键”，在这里选择视频 3，再选择“亮度信息”。播放看一下。

## 添加音效增强效果

还可以加一个音效，效果会更好一些。把这个音效拖拽到这里，它是一个水滴音效。按 Alt 键复制一份到这里。我也会连同这个水滴音效一块分享给大家。
