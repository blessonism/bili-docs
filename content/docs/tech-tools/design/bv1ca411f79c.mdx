对比在使用对比原则时
如果两个元素不完全相同
它们之间微小的差异
会让人觉得设计出现了问题
要想实现有效的对比碰撞
就必须使其截然不同

大与小的碰撞使文字的主次关系更容易分辨
不同字体的碰撞更能强调文案的气质和内容
色彩间的碰撞
使重要的内容得以凸显
