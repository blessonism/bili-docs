但是音频失真了
还夹杂着电流声
期待呼唤英雄百姓盼望平安

点赞我教你啊
首先我们要在pr效果窗口搜索音高换档器
把它拖到音频上
这时我们发现效果控件窗口出现的音高换挡器
点击编辑预设
我们选择默认勾选上高精度
接下来就需要用到我们小学2年级的数学公式
视频原调等于一除以加速倍数
因为这一段音频我加速了1.3倍
所以用1÷1.3
约等于0.7692
再把这个数值填到比率这里就大功告成了
我们听一下效果
